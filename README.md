# GreenSphere AI 🌍

GreenSphere AI is a full-stack environmental monitoring platform that provides real-time AQI monitoring, pollutant analysis, and live location-based environmental insights.

---

# Tech Stack

## Frontend

* React.js
* Tailwind CSS
* React Leaflet
* Recharts

## Backend

* Flask
* Python

## APIs

* OpenAQ API
* Geolocation API
* OpenStreetMap

---

# Features

* Real-time AQI monitoring
* PM2.5, PM10, NO2 analysis
* Live location detection
* Interactive AQI map
* Dynamic AQI status system
* AQI sharing and refresh
* Responsive modern UI

# Installation

## Clone Repository

```bash
git clone https://github.com/Priyamaurya2467/GreenSphere_AI.git
```

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

---

# Environment Variables

Create `.env` file:

```env
OPENAQ_API_KEY=your_api_key
```
